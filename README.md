# 🌐 Personal Portfolio Website Documentation (HTML & CSS)

## 📌 Project Description

This project is a Personal Portfolio Website built using **HTML and CSS**.
The website is designed to present personal information, skills, and contact details in a structured and visually appealing way.

It follows modern web design principles using semantic HTML and clean CSS styling.

---

## 🛠️ Technologies Used

* **HTML5**

  * Semantic elements like `<header>`, `<nav>`, `<section>`, `<footer>`
  * Form elements for contact section

* **CSS3**

  * Layout styling
  * Flexbox for navigation alignment
  * Styling for sections, buttons, and forms

---

## ⚙️ Setup Instructions

1. Install a code editor (e.g., VS Code)
2. Create a project folder named `Portfolio`
3. Create two files:

   * `index.html`
   * `style.css`
4. Copy the HTML code into `index.html`
5. Copy the CSS code into `style.css`
6. Link CSS in HTML using:

```id="shmdjw"
<link rel="stylesheet" href="style.css">
```

7. Open `index.html` in a browser

## 🧱 HTML Structure Explanation

### 1. Header Section

* Contains the website title and navigation menu
* Uses `<header>` and `<nav>` for semantic meaning

### 2. Navigation

* Links to different sections of the page
* Uses anchor links (`#about`, `#skills`, `#contact`)

### 3. About Section

* Displays profile image and introduction
* Uses `<section>` with headings and paragraph

### 4. Skills Section

* Displays list of skills
* Uses `<ul>` and `<li>` for structured listing

### 5. Contact Section

* Contains form elements:

  * Name input
  * Email input
  * Message textarea
* Uses HTML validation attributes like `required`

### 6. Footer

* Displays copyright information

---

## 🎨 CSS Styling Explanation

### 1. Global Styling

* Reset margins and padding using `*` selector
* Apply consistent font family

### 2. Header Styling

* Background color and text alignment
* Navigation links styled with spacing and colors

### 3. Layout Design

* Sections styled with:

  * Margin
  * Padding
  * Border radius
  * Box shadow

### 4. Image Styling

* Circular profile image using `border-radius: 50%`
* Centered using `display: block` and margin

### 5. Form Styling

* Inputs and textarea styled with padding and borders
* Button styled with hover effects

### 6. Resume Button

* Styled as a clickable button using background color and padding
